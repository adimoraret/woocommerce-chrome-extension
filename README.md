# README #

WooCommerce Goolge Chrome Extension - Work in Progress


### Summary ###

* Install
* Run
* Build & Deploy
* Miscellaneous

### Install ###
* Install npm and then run the following command
```
#!bash
npm install
```

### Run ###
* Run start
```
#!bash
npm start
```

### Build & Deploy ###
* Run build task
```
#!bash
npm run build
```
* Run build task silent
```
#!bash
npm run build -s
```

### Miscellaneous ###
* In order to debug ES6 code, we need to generate map files: 
```
#!bash
npm run deploy-mapfiles
```
